Kippt Reading Companion (KRC)
============

Built off of [ft-extension](https://github.com/kharmabum/ft-extension) template. How-to [install](https://developer.chrome.com/extensions/getstarted.html#unpacked).

Keyboard shortcuts:
  - default: Ctrl+Shift+K
  - mac: Command+Shift+K
